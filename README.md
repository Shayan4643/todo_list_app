# TODO List App

**Project**

- **Name:** TODO List App
- **Purpose:** A small web app to add, check off, and remove tasks using plain HTML, CSS, and JavaScript.

**Overview**

- **Description:** A minimal, client-side TODO list implemented as a learning exercise. Tasks are added via an input, can be marked complete, and deleted.
- **Stack:** HTML, CSS, JavaScript (no build tools or external dependencies).

**Usage**

- **Open:** `Open the app by opening` [index.html](index.html) `in your browser.`
- **Add Task:** Type a task into the input and press Enter or click Add.
- **Complete Task:** Click the checkbox next to a task to mark it done.
- **Delete Task:** Click the delete button/icon to remove a task.

**Files**

- **HTML:** [index.html](index.html) — app structure and markup.
- **Styles:** [style.css](style.css) — all styling for the UI.
- **Script:** [script.js](script.js) — DOM logic for adding, toggling, and removing tasks.

**Development**

- **Edit:** Modify [script.js](script.js) to change behavior or persistence.
- **Styling:** Update [style.css](style.css) for layout or theme changes.
- **Test:** Reload [index.html](index.html) after edits to verify changes.

**Next Steps / Ideas**

- **Persist tasks:** Add localStorage so tasks persist across reloads.
- **Edit tasks:** Add inline editing of existing tasks.
- **Filter:** Add filters for All / Active / Completed tasks.
